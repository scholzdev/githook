# git.ghook

Git operations and context.

## Snippets

### @no_large_files

Block files over 10MB:

```javascript
@no_large_files
```

### @conventional_commits

Enforce conventional commit format:

```javascript
@conventional_commits
```

### @branch_protection

Protect main/master branches:

```javascript
@branch_protection
```
